## 安装oh-my-zsh {#安装oh-my-zsh}

`Oh My Zsh` 是一种生活方式。它基于 `zsh` 命令行，提供了主题配置，插件机制，已经内置的便捷操作。给我们一种全新的方式使用命令行。

通过下面的命令进行安装，这个会把oh-my-zsh clone在 ~/目录下面

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

oh-my-zsh会生成自己的 `~/.zshrc` ，因此我们自己之前的 `.zshrc` 里面的内容我全部移动到 `~/.zsh/zshrc` 里面去了，在 `.zshrc` 里面来加载我们自己的 `zshrc` 就可以了，最后一行加上下面的代码就可以了 `source ~/.zsh/zshrc`


## [powerlevel10k安装](https://github.com/romkatv/powerlevel10k) {#powerlevel10k安装}

这个是我目前用起来最舒服zsh的主题了，由于上面我们安装了oh-my-zsh，因此选择下面的安装方式

```sh
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k
```

然后在~/.zshrc下面设置主题

```sh
ZSH_THEME="powerlevel10k/powerlevel10k"
```

然后重新打开一个终端，会自动进行配置，或者 `p10k configure`

安装powerlevel10k的时候会提示我们安装字体，选择安装就是了，然后重新启动iterm，会自动进行重新配置

### 安装自动补全插件 {#安装自动补全插件}

zsh本身是带了一定的自动补全的

\`\`\`jsx
brew install zsh-completions
\`\`\`

下面的代码放到我们的zshrc里面

\`\`\`jsx
if type brew &>/dev/null; then
    FPATH=$(brew --prefix)/share/zsh-completions:$FPATH

    autoload -Uz compinit
    compinit
  fi
\`\`\`

执行下面的命令

\`\`\`jsx
rm -f ~/.zcompdump; compinit
chmod go-w '/usr/local/share'
\`\`\`
